Class Routine API

Welcome to the Class Routine API! This API allows you to access class routine information for Thapathali Campus, which includes the following features:

  • Endpoint to retrieve the class routine for a specific class
  • Endpoint to retrieve the class routine for a specific teacher

API Endpoints

The Class Routine API has the following endpoints:

  • GET /api/routine/class - retrieves the class routine for the specified class for specified day according to the query parameters.
    • {% comment %}
  • GET /api/student/{studentId}/routine - retrieves the class routine for the specified student
    • {% endcomment %}
  • GET /api/routine/teacher - retrieves the class routine for the specified teacher for whole week according to the query parameters.

API Usage

To use the Class Routine API, you will need to send an HTTP GET request to one of the endpoints listed above. You may include query parameters as needed to specify the exact information you want to retrieve. For example: 

GET /api/routine/class?level=bei&day=thursday&semester=II

This request will retrieve the class routine for BEI semester II for Thursday. The API will return the data in JSON format. Please refer to the API documentation for more information on the available query parameters and the structure of the returned data. 

GET /api/routine/teacher?level=bei&teacher=ABC&semester=II&day=thursday

GET /api/routine/teacher?level=bei&teacher=ABC&semester=II

This request will retrieve the class routine for BEI semester II for whole week or thursday respectively of a specific teacher. The API will return the data in JSON format.
